#	Polyhedron Database:
#	There is a file for each solid with an index number given below.
#	The format is nearly obvious; the details are given in poly.man
#	(troff/nroff input). A lot of solids have symbolic data as well as numeric data.
#	The numeric data is normally accurate to about 1e-11.
#	The solids include the regular polyhedra (0-8), Archimedean (9-21),
#	some prisms and anti-prisms (22-31), Archimdean duals (32-44), the Johnson
#	solids (all convex polyhedra with regular faces, 45-136) and some weird
#	solids (137-141).
#	Please send errors, omissions and suggestions to research!andrew.
